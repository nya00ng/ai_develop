# AI Web App Starter: 미디어 서비스 레이아웃

## 프로젝트 개요

이 프로젝트는 프론트엔드 입문자를 위한 과제물로, HTML과 CSS만 사용해 구현한 미디어 콘텐츠 서비스 메인 페이지 뼈대입니다.  
시맨틱 태그(`header`, `main`, `section`, `article`, `footer`)를 활용해 구조적 의미와 접근성을 함께 고려했습니다.

## 주요 특징

- 시맨틱 마크업 준수
- 의미 있는 클래스명 사용으로 유지보수성 고려
- 추후 카드 확장 및 반응형 적용이 쉬운 CSS 구조

## 기술 스택

- HTML5
- CSS3

## 개발 로그

- **사용 도구:** Cursor + Claude API
- **요청 내용:** 미디어 서비스 메인 화면 Skeleton 생성
- **사용 프롬프트 요약 (1~4):**
  1. 현재 구조는 유지하고 카드 섹션이 더 명확히 보이도록 HTML 구조 정리
  2. 더 단순하게 변경
  3. 초보자 과제 수준에 맞춰 텍스트/구조 정리
  4. 시맨틱 태그 기준으로 다시 점검
- **결과:** 헤더/배너/카드/푸터 구조 생성 성공
- **보완점:** 카드 섹션 구조를 더 명확히 수정 예정

## 3줄 보고서

1. 오늘 Cursor와 Claude API를 연결하고 첫 HTML Skeleton을 생성했습니다.
2. "무엇을 만들지 + 어떤 방식으로 만들지"를 같이 써야 결과가 더 좋아진다는 점을 확인했습니다.
3. 목요일 전까지 카드 섹션 구조를 더 명확하게 정리해보겠습니다.

## 실행 방법

### 방법 1) 파일 직접 열기

1. 프로젝트 폴더에서 `index.html` 파일을 찾습니다.
2. 파일을 더블클릭하거나 브라우저로 드래그해서 엽니다.

### 방법 2) 로컬 서버로 실행 (권장)

1. 터미널에서 프로젝트 폴더로 이동합니다.
2. 아래 명령어를 실행합니다.

```bash
python -m http.server 5500
```

3. 브라우저에서 [http://localhost:5500](http://localhost:5500) 로 접속합니다.
