# 명언 생성 웹 페이지

입문자 학습용으로 만든 바닐라 JS 프로젝트입니다.  
`index.html`, `style.css`, `app.js` 3파일 구조로 구성되어 있으며, 랜덤 명언 표시와 사용자 입력 명언 추가, 즐겨찾기, 테마 전환 기능을 제공합니다.

## 프로젝트 구조

```text
02_AI01/
├─ index.html   # 화면 구조, 버튼/폼/접근성 속성
├─ style.css    # 모바일 퍼스트 UI, 카드 디자인, 애니메이션, 테마 스타일
└─ app.js       # 명언 로직, localStorage 저장/복원, 이벤트 처리
```

## 구현 기능 정리

- **랜덤 명언 출력**
  - 기본 명언 21개 배열 제공
  - `다음 명언` 클릭 시 랜덤 명언/저자 표시
  - 직전 명언과 같은 항목이 연속 선택되지 않도록 처리

- **명언 복사**
  - `복사하기` 버튼으로 현재 명언을 클립보드에 복사
  - 성공/실패 메시지를 상태 영역에 안내

- **즐겨찾기**
  - `즐겨찾기` 버튼으로 현재 명언 저장/해제 토글
  - 버튼의 `aria-pressed` 상태 동기화
  - 즐겨찾기 개수 실시간 표시
  - 즐겨찾기 데이터는 `localStorage`에 저장되어 새로고침 후 유지

- **라이트/다크 테마 선택**
  - `라이트`, `다크` 버튼으로 테마 직접 선택
  - 선택 상태 버튼 강조(`active`) 및 접근성 속성 반영
  - 테마 설정은 `localStorage`에 저장되어 재접속 시 복원

- **내 명언 추가**
  - 폼에서 **명언 + 저자** 직접 입력 후 `명언 추가`
  - 입력 유효성 검사
    - 빈 값 방지
    - 명언 최소 5자
    - 저자 최소 2자
  - 추가 즉시 화면 반영
  - 사용자 명언은 `localStorage`에 저장되어 새로고침 후 유지

- **UI/UX 및 접근성**
  - 모바일 퍼스트 반응형 레이아웃
  - 중앙 정렬 카드형 디자인
  - 버튼 `hover`/`focus-visible` 스타일
  - 상태 메시지 영역(`aria-live`)으로 동작 피드백 제공
  - 명언 전환 시 페이드 애니메이션 적용

## localStorage 키

- `quote-app.favorites`: 즐겨찾기 명언 배열
- `quote-app.theme`: 현재 테마(`light` 또는 `dark`)
- `quote-app.myQuotes`: 사용자가 추가한 명언 배열

## 실행 방법

1. 프로젝트 폴더(`02_AI01`)를 엽니다.
2. `index.html`을 브라우저에서 실행합니다.
3. 버튼/폼을 사용해 기능을 테스트합니다.

## 학습 포인트

- DOM 선택 및 이벤트 처리
- 배열 기반 랜덤 로직
- `localStorage`를 활용한 브라우저 데이터 영속화
- 접근성(ARIA)과 키보드 포커스 스타일 적용
- 모바일 퍼스트 CSS 구성 방법
